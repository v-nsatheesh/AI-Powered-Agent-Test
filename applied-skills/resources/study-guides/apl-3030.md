# Study guide for the Applied Skills assessment lab: Automate Azure Load Testing by using GitHub Actions

## Purpose of this document

This study guide provides a summary of the topics covered in the
assessment lab, along with additional resources to help you prepare.
Note that the learning path on the credential details page may include
more modules than the "Tasks performed" to provide a cohesive learning
experience.

### Tasks at a glance

#### Prepare the GitHub and Azure environments

-   Create an identity, including user-assigned managed identity or a service principal

-   Configure authorization for the identity

-   Connect GitHub to the Azure subscription

-   Configure an Azure App Service deployment slot

    Supporting module (s):

    -   [Implement Azure Load Testing](https://learn.microsoft.com/training/modules/implement-azure-load-testing/)

#### Configure GitHub Actions to deploy an app to Azure App Service

-   Create and configure a deployment workflow

-   Implement deployment from a dev branch to Azure App Service

    Supporting module (s):

    -   [Implement GitHub Actions for Azure](https://learn.microsoft.com/training/modules/implement-github-actions-azure/)
	
	-   [Implement GitHub Actions](https://learn.microsoft.com/training/modules/implement-github-actions/

#### Create and configure load tests

-   Create an Azure Load Testing resource

-   Create a load test, including URL-based and JMX

-   Configure failure criteria for a load test

    Supporting module (s):

    -   [Implement Azure Load Testing](https://learn.microsoft.com/training/modules/implement-azure-load-testing/)

#### Configure GitHub Actions to run load tests

-   Create and configure a load testing workflow

-   Upload artifacts

    Supporting module (s):

    -   [Implement Azure Load Testing](https://learn.microsoft.com/training/modules/implement-azure-load-testing/)

#### Manage deployment slots and pull requests by using GitHub Actions

-   Automate pull request approvals

-   Swap staging and production slots in Azure App Service

-   Implement triggers and dependencies in GitHub workflows

    Supporting module (s):

    -   [Implement Azure Load Testing](https://learn.microsoft.com/training/modules/implement-azure-load-testing/)
	
	-   [Implement GitHub Actions](https://learn.microsoft.com/training/modules/implement-github-actions/)
