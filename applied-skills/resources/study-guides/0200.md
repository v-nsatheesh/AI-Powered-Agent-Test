# Study guide for the Applied Skills assessment lab: Get started with classes, properties, and methods in C#

## Purpose of this document

This study guide provides a summary of the topics covered in the
assessment lab, along with additional resources to help you prepare.
Note that the learning path on the credential details page may include
more modules than the "Tasks performed" to provide a cohesive learning
experience.

### Tasks at a glance

#### Create classes, objects and types

-   Choose between classes, objects, and types

-   Create classes

-   Create objects

-   Use types

-   Use namespaces

    Supporting module (s):

    -   [Get started with classes and objects](https://learn.microsoft.com/training/modules/get-started-classes-objects/)

#### Work with properties

-   Create properties

-   Control access to properties

-   Use different property types (nullable / default values)

    Supporting module (s):

    -   [Implement class properties and methods](https://learn.microsoft.com/training/modules/implement-class-properties-methods/)

#### Work with methods

-   Create methods

-   Use Method return options

-   Control access to methods

-   Overload methods

    Supporting module (s):

    -   [Implement class properties and methods](https://learn.microsoft.com/training/modules/implement-class-properties-methods/)

#### Call properties and methods

-   Instantiate classes

-   Use class properties

-   Use class methods

-   Use static classes

-   Call methods and handle return values

    Supporting module (s):

    -   [Manage class implementations](https://learn.microsoft.com/training/modules/manage-class-implementations/)
