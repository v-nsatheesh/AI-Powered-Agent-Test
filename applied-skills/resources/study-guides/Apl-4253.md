# Study guide for the Applied Skills assessment lab: Get started with Azure management tasks

## Purpose of this document

This study guide provides a summary of the topics covered in the
assessment lab, along with additional resources to help you prepare.
Note that the learning path on the credential details page may include
more modules than the "Tasks performed" to provide a cohesive learning
experience.

### Tasks at a glance

#### Perform basic subscription tasks

-   Manage and view Azure settings

-   Add locks to resources

-   Add tags to resources

-   View subscription level settings

    Supporting module (s):

    -   [Describe Microsoft Azure resources management](https://learn.microsoft.com/training/modules/describe-microsoft-azure-resources-management/)

#### Perform basic network services tasks

-   View subnets and related resources

-   Gather virtual network discovery

-   View NSGs

-   Assign an NSG

    Supporting module (s):

    -   [Describe Microsoft Azure resources management](https://learn.microsoft.com/training/modules/describe-microsoft-azure-resources-management/)

#### Perform basic virtual machines tasks

-   Configure basic virtual machine settings

-   Attach a new disk

-   Change the size of a virtual machine

-   Configure the virtual network interface

-   Gather virtual machine inventory

-   Start and stop a virtual machine

    Supporting module (s):

    -   [Describe Microsoft Azure resources management](https://learn.microsoft.com/training/modules/describe-microsoft-azure-resources-management/)

#### Perform basic storage tasks

-   Manage access keys

-   Get connection information

-   Create shared access signatures

-   Configure file share settings

-   Upload a file to Azure Files

-   Create a Blob storage container

-   Upload a file to a Blob storage container

    Supporting module (s):

    -   [Describe Microsoft Azure resources management](https://learn.microsoft.com/training/modules/describe-microsoft-azure-resources-management/)
