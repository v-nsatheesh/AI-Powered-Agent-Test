# Study guide for the Applied Skills assessment lab: Develop generative AI apps with Azure OpenAI and Semantic Kernel

## Purpose of this document

This study guide provides a summary of the topics covered in the
assessment lab, along with additional resources to help you prepare.
Note that the learning path on the credential details page may include
more modules than the "Tasks performed" to provide a cohesive learning
experience.

### Tasks at a glance

#### Prepare the development environment

-   Deploy an Azure OpenAI Service model

-   Retrieve and store an endpoint and a key

-   Create a new console application

    Supporting module (s):

    -   [Build your
        kernel](https://learn.microsoft.com/training/modules/build-your-kernel/)

#### Run prompts with Semantic Kernel

-   Create a Semantic Kernel object

-   Create and run a prompt

    Supporting module (s):

    -   [Build your
        kernel](https://learn.microsoft.com/training/modules/build-your-kernel/)

    -   [Create plugins for semantic
        kernel](https://learn.microsoft.com/training/modules/create-plugins-semantic-kernel/)

#### Improve the prompt 

-   Refine the prompt to be more specific, such as changing the output
    format or the style

-   Define message roles in chat completion prompts

-   Add variables to the prompt

-   Add context to the prompt by using the ChatHistory object

-   Configure few-shot prompting

-   Configure zero-shot prompting

-   Use the Handlebars prompt template

    Supporting module (s):

    -   [Create plugins for semantic kernel](https://learn.microsoft.com/en-us/training/modules/create-plugins-semantic-kernel/)

#### Create native plug-ins for Semantic Kernel 

-   Create native plug-ins for Semantic Kernel

-   Create and decorate a plug-in class

-   Invoke native function manually

-   Invoke native function automatically

-   Configure function choice behavior

    Supporting module (s):

    -   [Give your AI agent
        skills](https://learn.microsoft.com/training/modules/give-your-ai-agent-skills/)

#### Implement filters on invoked functions 

-   Apply trust filters on user input

-   Apply trust filters on function output

-   Implement function filters

    Supporting module (s):

    -   [Combine Prompts and
        Functions](https://learn.microsoft.com/training/modules/combine-prompts-functions/)
