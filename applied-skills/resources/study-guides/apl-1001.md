# Study guide for the Applied Skills assessment lab: Deploy containers by using Azure Kubernetes Service

## Purpose of this document

This study guide provides a summary of the topics covered in the
assessment lab, along with additional resources to help you prepare.
Note that the learning path on the credential details page may include
more modules than the "Tasks performed" to provide a cohesive learning
experience.

### Tasks at a glance

#### Deploy an Azure Kubernetes Service (AKS) cluster 

-   Deploy an AKS cluster into an existing virtual network

-   Create a Windows node pool

    Supporting module (s):

    -   [Deploy an Azure Kubernetes Service
        cluster](https://learn.microsoft.com/training/modules/deploy-azure-kubernetes-service-cluster/)

#### Deploy and use Azure Container Registry (ACR)

-   Create an Azure Container Registry (ACR)

-   Build a container image for a Linux app and upload it to a
    designated repo in the ACR

-   Build a container image for a Windows app and upload it to a
    designated repo in the ACR

    Supporting module (s):

    -   [Deploy and use Azure Container
        Registry](https://learn.microsoft.com/training/modules/deploy-use-azure-container-registry/)

#### Configure an Azure Kubernetes Service (AKS) cluster

-   Apply cluster configuration by using Azure Policy

-   Configure infrastructure encryption by using Azure CLI

-   Create custom namespaces

    Supporting module (s):

    -   [Configure an Azure Kubernetes Service
        cluster](https://learn.microsoft.com/training/modules/configure-azure-kubernetes-service-cluster/)

#### Deploy applications to Azure Kubernetes Service (AKS)

-   Apply container and pod settings by using Azure Policy

-   Deploy an app based on the first container image to the Linux pool
    and the first namespace

-   Deploy an app based on the second container image to the Windows
    pool to the second namespace

-   Configure storage for applications that run on AKS

    Supporting module (s):

    -   [Deploy applications to Azure Kubernetes
        Service](https://learn.microsoft.com/training/modules/deploy-applications-azure-kubernetes-service/)

#### Configure scaling in Azure Kubernetes Service (AKS)

-   Implement horizontal pod autoscaling

-   Implement cluster virtual nodes by using ACI

-   Deploy pods to virtual nodes

    Supporting module (s):

    -   [Configure scaling in Azure Kubernetes
        Service](https://learn.microsoft.com/training/modules/configure-scaling-azure-kubernetes-service/)
