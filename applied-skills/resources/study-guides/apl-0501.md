# Study guide for the Applied Skills assessment lab: Get started with identities and access using Microsoft Entra

## Purpose of this document

This study guide provides a summary of the topics covered in the
assessment lab, along with additional resources to help you prepare.
Note that the learning path on the credential details page may include
more modules than the "Tasks performed" to provide a cohesive learning
experience.

### Tasks at a glance

#### Perform basic user management tasks

-   Create a new user

-   Add a license to a user

-   Invite an external user

-   Assign a role to the user

-   Bulk import users

    Supporting module (s):

    -   [Create, configure, and manage identities](https://learn.microsoft.com/training/modules/create-configure-manage-identities/)

#### Perform basic group management tasks

-   Create a Microsoft 365 group

-   Create a security group

-   Add an existing user to a new group

-   Add licenses, owners, and members to a group

    Supporting module (s):

    -   [Create, configure, and manage identities](https://learn.microsoft.com/training/modules/create-configure-manage-identities/)

#### Perform basic password protection tasks

-   View password protection lockout settings, including duration and threshold

-   View banned passwords, including adding banned passwords

    Supporting module (s):

    -   [Describe the authentication capabilities of Microsoft Entra ID](https://learn.microsoft.com/training/modules/explore-authentication-capabilities/)

#### Perform basic self-service password reset (SSPR) tasks

-   View SSPR properties for a specific group

-   View SSPR authentication methods

-   View SSPR registration

-   View SSPR notifications

    Supporting module (s):

    -   [Describe the authentication capabilities of Microsoft Entra ID](https://learn.microsoft.com/training/modules/explore-authentication-capabilities/)

#### Perform basic multifactor authentication tasks

-   Enable or disable per-user MFA settings

-   View trusted IPs for MFA

-   View account password protection lockout settings

-   View service settings

    Supporting module (s):

    -   [Describe the authentication capabilities of Microsoft Entra ID](https://learn.microsoft.com/training/modules/explore-authentication-capabilities/)

#### Perform basic conditional access policy tasks

-   Review a conditional access policy

-   Perform What If analysis for a conditional access policy

-   Enable or disable a conditional access policy

    Supporting module (s):

    -   [Describe access management capabilities of Microsoft Entra](https://learn.microsoft.com/training/modules/explore-access-management-capabilities/)
